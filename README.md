Akib Al Imran — Portfolio

🔗 Live site:[ akibalimran.vercel.app](https://akibalimran.vercel.app/)

My personal portfolio — a full-stack + AI/ML + mobile + embedded systems story in one place, instead of a generic resume page. Built to actually show how I work: real stack details, live demos, and verifiable certificates instead of static screenshots.

What's inside


About — career objective, quick facts, current focus
Education — full academic timeline with clickable, verifiable certificates
Skills — grouped by category across languages, frontend, backend, mobile/AI-ML, data, and tools
Projects — 11 projects spanning web, mobile, ML, and robotics, each with live demo + GitHub links
Experience — internship history with linked completion certificates
Services — what I can build for teams and clients
Contact — direct message form (EmailJS) for roles, internships, or freelance work


Tech stack


TanStack Start + React 19
Tailwind CSS v4 + shadcn/ui
Vite + Nitro
EmailJS for the contact form
TypeScript throughout


Running locally

bashnpm install
npm run dev

Build for production:

bashnpm run build
